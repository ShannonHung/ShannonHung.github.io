1 題目描述


給定一個二元樹，返回其節點值的層序遍歷。換句話說，從左到右，逐層遍歷所有節點。

2 解法

這題很明顯，我們可以使用BFS的方式來解，因為BFS是一種逐層遍歷的方式。

我們會需要

  • next_level 來存放下一層的節點，然後透過呼叫自己，放入next_level的節點。
  • nodes 是當層的所有節點。
  • values 是用來存放當層的節點值。
  • result 就是結果。
