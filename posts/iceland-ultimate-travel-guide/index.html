冰島自由行2024冬天旅遊指南: 行程規劃/保險/自駕注意

前言

在德國讀雙聯終於考完期末考時，那時是二月初。我們想著要怎麼規劃一個月的寒假，剛好小夥伴們在我家討論想去的國家時，大家都有相同的願望就是去冰島看極光，剛好極光最適合觀賞的月份是10月~3月，因此我們就開啟了Flyscanner搜尋哪一天的機票最便宜，看了看發現2/26的去程跟3/7回程最便宜，二話不說就訂機票了，訂完機票我們就會有動力來開始計劃11天10夜的行程！

那這篇文章主要記錄了當時我們的行程規劃、預算規劃、自駕注意事項、保險、意外事件等等，希望對大家有幫助！

我先來說一下結論，冰島整趟旅遊最值得的部分：

  1. 自駕環島，從離開城市開始，進入彷彿另一個星球的世界，沿途的風景令人不勝讚嘆
  2. 極光（一定要先做好拍攝功課，因為極光不等人）
  3. 參加冰川健行與藍冰洞的跟團（藍冰洞只有冬天才有）
  4. 雪地遇難的經驗（這是最印象深刻的，但當然以安全為前提，要做好雪地自駕的功課還有遇難的應對措施和保險）

01-行程安排

我們是四人行，在交通方面非常推薦自駕冰島，但是一定要四輪驅動，因為有很多地方是只有四驅車才能進入，否則非四驅車進入保險會失效。行程安排大概如上圖，按照紅橙黃綠藍紫的顏色代表不同天的行程。

Day01 取車+小豬超市準備食物

這天因為飛機下午才抵達冰島，因此我們主要就是取車、買食物、住宿。

Day02 火口湖 + 間歇泉 + 斯里蘭卡瀑布

我就總結一下，這個行程最印象深刻最美好的：

  • 我最印象深刻的是賽裡雅蘭卡瀑布，因為我們把它放在最後一個行程，天氣好加上遇到夕陽落下時刻，當夕陽打在瀑布上和黃綠色的草地上，真的是太美了！
  • 第二個很推的是飯店Guesthouse Hateigi，他是我在冰島住到最棒的民宿，浴室跟廚房都在房間內，非常方便不需要跟別人搶，同時他還在浴室附上了洗衣機，還有洗碗機！真的是什麼都有！

Day03 彩虹瀑布 + 迪拉半島 + 黑沙灘 + 紅頂教堂 + 羽毛河峽谷

這裡就先不提供時間，因為時間有限所以不是google map上的景點都有去，但是主要下面表格是我們最後有去的地點。

  • 最印象深刻的應該是彩虹瀑布，非常推薦走旁邊的步道上瀑布上面，景觀很壯觀！然後瀑布真的很容易看到彩虹，建議早上去，因為陽光剛好可以照到瀑布才會有彩虹。
  • 第二名是迪拉半島，會先開車抖上到一個坡度，沿途也很壯觀，從這裡看黑沙灘很漂亮！但是黑沙灘真的太多人了，加上玄武岩小小的，所以印象不太好。
  • 第三名是羽毛和峽谷，一次身處在一整片的草原跟山谷還是很壯觀！有時候還會不小心看到野生的鹿群在遠方奔跑！

Day04 杰古沙龍冰河湖 + 鑽石沙灘 + 羊排晚餐

這天的行程是最Chill的一天，最印象深刻的應該就是吃羊排跟看鑽石沙灘，還有看極光

  • 鑽石冰沙灘 真的很美，沒有看過冰塊在沙攤上，很建議盡量走到裡面靠近岩牆的地方，因為那裡人比較少，記得撿一塊冰塊跟海合照一張。
  • Pakkhus Restaurant 這間餐廳的羊排跟奶油龍蝦還不錯，但價位偏高，羊排吃不出來羊味，奶油龍蝦很新鮮，沒有到說非常好吃到難忘，但是是一個很好的體驗。
  • 霍芬旅館 這間飯店很乾淨，廚具應有盡有，最棒的是霍芬小鎮是很適合看到極光的地方，這個旅館後面就有一個小山丘，超級適合看極光！但是晚上爬上去完全沒有燈，要記得帶手電筒，盡量走靠柵欄的地方比較不那麼陡峭。

Day05 跟團：冰川健行 + 藍冰洞

今天的行程是最棒最難忘的因為3:30結束了冰川行程，我們原本住的飯店是Studio Apartments他是在93號公路的底端，路途夾在兩座山脈之間，剛好那天暴風雪…我們卡在93號公路的前端，所以無法順利抵達飯店，只好臨時改住新旅館。硬要說的話，這天最印象深刻的是：

  • 冰川與藍冰洞健行，我們做了非常多功課，因為難得冬天自駕，所以當然要看只有冬天才看得到的藍冰洞！夥伴多方的比價後，最後選擇了這個跟團，他們不僅會有冰川健行，還會去最大的藍冰洞，價位在萬元內是可以負擔的！這個行程導遊也說是他最喜歡的行程！
  • 雪地自駕遇難，這個真的很令人印象深刻，所以為此我會為這個部分特別寫應該怎麼應對的方法。

Day06 米湖 + 惠爾山

經歷了昨天的暴風雪，今天的行程就特別注意不要拉車到傍晚才回去，所以今天主要只去一個景點，而其他景點像是玄武岩走廊因為太遠所以不去、賽爾佛斯瀑布和Krafla火山口＋Víti火山湖則是因為前一天的暴風雪無法通行所以也取消不去。而米湖也因為前一天的暴風雪，覆蓋一層雪，根本看不出來是湖…

最印象深刻的是：

  • 超級棒的飯店，大推Myvatn Berjaya Iceland Hotels，窗戶看出去的景色非常好看！後面也很適合拍極光（但是教堂有點太亮而已），飯店提供的洗髮乳跟沐浴乳都很頂級、還提供雪鏟跟刮除冰的工具。
  • 惠爾山也很不錯，爬山山頂不會很久，但是因為前一天暴風雪的緣故，導致上山跟下山都有冰跟雪，容易滑倒…，但是山上看出去一望無際的景色還是令人讚嘆！
  • 極光大爆發，晚上的溫度低到-16度，但是天氣極好，並且極光大爆發！

Myvatn Berjaya Iceland Hotels旅館 沒廚房，但是設備和景色一級棒，房間內提供不錯的膠囊咖啡、沐浴乳、洗髮乳和潤髮乳等，非常好的室內設計！

Day07 眾神瀑布

接下來這天的行程，從眾神瀑布到阿克雷里，會經過全境唯一收費的隧道，他沒有收費站、沒有柵欄只有一塊提示牌，看其他攻略說3小時內繳費就可以了，這條隧道的收費是Kr.1650，但是我們沒有走收費隧道，有免費公路83號84號可以走，繞開風景很漂亮，大概多開16公里，但是可能會因為天氣惡劣而導致關閉，風挺強的，當時開的時候雪在公路上飄，所以看不太清楚地面的樣子。

隧道位置圖片示意圖


當天印象比較深刻的部分是旅館，這間旅館身處在有點像沖積平原的地方，可以看到很多河流把河床沖刷成三角狀，而飯店處在一個偏遠的地方，附近完全沒有光害！可惜那天的天氣雲層多，否則那天是我們來旅行極光KP值最高的一天！如果有機會，天氣好一定要來住住看！風景也很好，廚房跟公共空間給人一種溫馨的感覺。

這天的行程可以說是環島的最後一天，因為開車時間長達6小時，為了不要太晚回到雷克雅維克，所以我們只去了幾個簡單的景點，順路的草帽山跟看海豹。

Day09 藍湖溫泉 + 市區燈塔追光

藍湖溫泉真的很棒！應該是人生中難忘的溫泉泡湯經驗吧…，雖然台灣的溫泉也不錯，但是這裡的溫泉可以在泡湯的同時欣賞風景、喝飲品、敷面膜的體驗還是第一次。我認為如果你要去泡藍湖溫泉有幾個建議：

  • 不需要帶毛巾、洗澡用品，他們會提供。
  • 可以不用買到Prime的票，買基本票就好，Prime多了一杯飲料跟比較好的面膜，但是基本的票也是有三溫暖跟桑拿可以使用，他們三溫暖不需要驗證就可以進去！

那天我們很幸運的遇到來自香港的朋友，他們免費送我們他們的手環可以用來兌換飲料，很幸運。

藍湖 需預約
開車 藍湖 to 市區 45min
市區燈塔 市區追極光最佳地點，但是我得說，人太多了！建議去找市區偏遠的地方，看極光還比較好…，蠻推薦這裡沒有光害。如果硬要在燈塔，只有一個意見那就是等到過12點，像是1~2點人少點再過去比較好，不然所有車子都卡在燈塔附近，車子很難出來就算了，車子的遠光燈都會影響觀賞體驗…還有一堆遊客會開閃光燈…
Hostel B47 有廚房，睡飽點 明天睡機場

Day10 寄明信片 + 市區採買

這天是我們最後一天，所以就是把一些小事情做完，像是寄明信片、採買一些冰島的特產，然後就是回家了。如果你預算不高，大概2000內可以買以下東西：

  • 明信片（去郵局買郵票寄信，寄回台灣大概100台幣內）
  • 羊毛磁鐵（本店大概一個300台幣左右）
  • 羊毛帽（大概一個1500台幣左右）
  • 去機場再買冰島巧克力，買Salt口味！超級好吃！

icemart 紀念品

Hertz還車的地點挺特別的，我們迷路了一下，後來發現他當初給我們鑰匙的信封袋背面有寫。然後他們的還車方式是，你把車停在還車場，然後把wifi機器留在車上，鑰匙放在還車場旁邊入口建築物門口的信箱口，那個信箱口的設計真的很白痴，他們是在牆上挖洞…你要把鑰匙投入洞裡，然後那個洞被信箱盒蓋住…。

Day11 回家

這天回家的時候，剛好遇到德國法蘭克福機場罷工+德鐵罷工…又是另一個故事了(´･ω･`)。 但是我們很幸運的，飛機沒有取消，改搭公車回Karlsruhe。

03-開銷花費

廢話不多說，先給結論，含來回機票、冰川健行等，所有的費用大概是1451歐元（這裡還涵蓋一些意外的費用，像是臨時住的旅館和救援費用的差額），整體11天10夜的開銷大概會需要抓45000台幣。

以下圖片的表格是個人花費喔！旁邊的細節總額是四個人的開銷。

吃的部分
我覺得我們自己煮的挺好的，平均一餐不到5歐元：

  • 早午餐：
    • 早上會把午餐一起做好，主要吃吐司、火腿、nutella巧克力醬、起司、水煮蛋，有時候可以配冰島的巧克力牛奶、或是燕麥奶（不用冰）。
    • 建議把夾鏈袋帶上，可以把吐司放在裡面
  • 晚餐
    • 有時候買超大條鮭魚吃，冰島一定要去小豬超市買大鮭魚超級划算又好吃！
    • 主要還有吃自己帶的泡麵
    • 如果沒有泡麵就吃義大利麵也不錯

去小豬超市採購的時候，主要必買的東西有：

  • 雞蛋：因為我們有四個人，所以都會確保車上會有兩盒雞蛋
  • 吐司、起司：做早午餐用
  • 玉米罐頭：因為我們會帶康寶濃湯
  • 蔬菜：大白菜真是可以久放的好蔬菜
  • 蘋果：不要忘了補充水果
  • 有時候想奢侈點時：
    • 燕麥奶
    • 豬肉片
    • 優格

住宿的部分
住宿主要會有幾點注意：

  • 可以先透過booking.com用地圖在你的行程附近找便宜的住宿有哪些
  • 然後記得透過google map查一下是否有更便宜的價位（有時候google會列出不同平台的價格），甚至可以找一下旅館是否有官網，有時候官網更便宜…
  • 接著要注意，如果你是冬天自駕，準備預定北部的飯店，如果不是大城市的話…
    • 注意飯店是否偏僻、遠離1號公路、夾帶山脈中間、或是查看路況的網站看一下通往該旅館的路是否可以通行...
    • 這點很重要，因為我們就遇到了暴風雪，我們的飯店就在93號公路的底端，路途夾在兩座山脈之間，無法順利抵達飯店，只好臨時改住新旅館。
    • 你也可以不訂旅館，當天看天氣狀況決定，因為冬天的旅館一定預定的到...
  • 建議選有廚房的飯店，才可以省錢自己煮（自己煮也比較好吃）

最大的開銷
最大個開銷主要有以下幾項：

  • 機票：如果是留學生，建議直接去Skyscanner選最便宜的日子去。
    • 啊！記得有時候直接去官網買比SkyScanner便宜超多！
  • 租車：可以查看04-自駕和租車的部分
  • 冰川健行：超級推薦這個跟團!冰川冰洞一次滿足！
  • 汽油費：通常租車公司會給加油卡，去合作的加油站比較便宜，然後不要使用台灣的信用卡刷…否則非N1的加油站每次會先預扣6000台幣手續費，很傷。但是如果你是使用歐洲的銀行帳戶像是Revolut，刷完就會立刻退回了。N1會比其他加油站還要貴…。
